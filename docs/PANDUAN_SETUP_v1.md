# Panduan Pemindahan Proyek (OFFLINE MODE) v1.0

Dokumen ini menjelaskan cara memindahkan proyek **Dashboard CV** agar bisa berjalan 100% tanpa internet di komputer lain.

## Prasyarat
- **Node.js** (versi 18 atau lebih baru)
- **PocketBase** (File database tunggal)

## Langkah-Langkah Setup
1. **Copy Proyek**: Salin seluruh folder proyek ini ke komputer/flashdisk.
2. **Download PocketBase**: Ambil file `pocketbase` dari pocketbase.io dan letakkan di folder utama.
3. **Jalankan**: Klik 2x file `MULAI_DASHBOARD_WINDOWS.bat` atau `MULAI_DASHBOARD_MAC.command`.
4. **Setup Database**: Ikuti instruksi di [SKEMA_DATABASE_v1.md](./SKEMA_DATABASE_v1.md).
